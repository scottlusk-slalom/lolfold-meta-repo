# Organizational Context Index

This directory holds organizational-level context that informs all projects and specs.

## Contents

| File | Purpose |
|------|---------|
| `index.md` | This file — map of organizational context |
| `standards.md` | Engineering standards, conventions, and non-negotiables |
| `patterns.md` | Approved architectural patterns and technology choices |
| `glossary.md` | Domain terminology and definitions |

## Usage

Agents reference this directory to ground decisions in organizational context. The context-curator agent promotes validated patterns and decisions from completed specs up to this level when they represent org-wide learning.
